学习内存池和内存对齐时，在知乎看到了这个算法，记录一下。

``` c
unsigned int getCeilPowerOfTwo(unsigned int num)
{
	//unsigned int 类型能表示的最大的 2 的 N 次方数就是 1U << 31。
	if(n > (1U << 31))	return (1U << 31);
	//如果 num 本身就是 2 的 N 次方数，直接返回原值。
	if(n == 0 || n & (n - 1) == 0)	return num;
	//将 num 的二进制数的最大及以下数位全部变成 1。
	num |= num >> 1;	//先让前 2 大数位变成 1。
	num |= num >> 2;	//再让前 4 大数位变成 1。
	num |= num >> 4;	//再让前 8 大数位变成 1。
	num |= num >> 8;	//再让前 16 大数位变成 1。
	num |= num >> 16;	//再让最后 16 个数位（如果有的话）变成 1。
	//最大及以下数位全是 1 的数再加 1，就是大于且最接近原值的 2 的 N 次方数了。
	return num + 1;
}
```

举例：
